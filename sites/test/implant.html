Какие хирургические процедуры проводятся в клинике регенеративно

  • Имплантация зубов в боковых участках для восстановления жевательной эффективности
  • Имплантация зубов во фронтальном участке для восстановления эстетики по линии улыбки.
  • Одноэтапная имплантация с немедленной нагрузкой.
  • Компьютерная 3D диагностика верхней и нижней челюсти, гайморовых пазух, ВНЧС перед установкой имплантов.
  • Имплантация зубов «Все на 4» для условно-съемного протезирования
  • Синус-лифтинг. Проводится перед имплантацией по показаниям при сильной атрофии кости в боковых участках верхней челюсти.
  • Костная пластика. Проводится по показаниям для увеличения объема костной ткани перед установкой имплантата.
  • Удаление зубов любого уровня сложности. Проводится по показаниям перед установкой имплантата
  • Расщепление альвеолярного гребня. Проводится по показаниям для создания благоприятных условий для установки имплантата

Жалобы:

Пациент обратился в клинику с повреждением зубов на обеих челюстях и полностью отсутствующими жевательными зубами на верхней челюсти. Зубы пациента были восстановлены применением метода базальной имплантации Basal Complex.

Диагноз:

хронический вторический карьес зубов 1.2, 1.1.

Лечение

Проведено удаление оставшихся зубов на верхней челюсти и весь зубной ряд восстановлен за 3 дня при помощи базальных имплантов. На нижней челюсти сильно поврежденные зубы также были удалены и установлен съемный зубной протез. Разрушенные зубы верхней челюсти удалены и восстановлены при помощи базальной имплантации. Концевой дефект по обеим сторонам нижней челюсти восстановлен сегментами имплантов.
